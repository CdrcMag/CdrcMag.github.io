Accueil

/ Pac Human

Pac Human est un jeu d'arcade développé pour Athéo Ingénierie.

/ Car Patrol

Car Patrol est un jeu mobile d'arcade / scoring, réalisé en partenariat avec Rec'im.

/ Splash

Splash est un serious game platformer en 2D développé sur Unity, visant à introduire le grand public au notions d'algorithmie.

/ Cyber Ball

Cyber Ball a été mon projet de 3ème année développé sur Unity. Le jeu a été pensé pour être joué avec la Kinect 2.0 sur une borne d’arcade.

/ Crash Wave

Crash Wave est une démo développée sur Unity en une semaine en Décembre 2019. C'est un jeu de course en multi-joueurs, utilisant la web-cam pour les déplacements. Les joueurs s'inclinent verticalement pour se déplacer. Aucun input clavier ou souris n'est requis.

/ Brewery Tycoon

Brewery Tycoon est une démo d'un serious game développé sur Unity. On y incarne un robot cherchant à commencer son business de brassage de bière en apprenant les étapes de base de la création de bière.

/ Sorciers Vs Vikings

Sorciers Vs Vikings est un visual novel développé sur Ren'py. Il visait à proposer une expérience narrative à plusieurs embranchements.
