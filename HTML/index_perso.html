Accueil

/ Repentance

Repentance est un platformer 2D développé durant la Global Game Jam 2020.

/ Octogon

Octogon est un jeu mobile de scoring au style minimaliste.

/ Micro Fun

Micro Fun est un jeu réalisé dans le cadre d'un défi ayant pour ambition de proposer de des pistes de reflexion rendant "fun" la mécanique d'appuyer sur un bouton.

/ Glow In Nightmare

Glow in nightmare est un jeu d'épouvante / rail shooter, développé pour la Mix And Game Jam 2020. Réalisé avec Bastien Prob.

/ Leaf

Leaf est un petit jeu de reflexes fait en game jam en Septembre 2020. Il a été réalisé avec l’ambition d’implémenter une structure globale modulable et l'application de divers design patterns.

/ Roll Boll

Roll Boll est un jeu mobile de type infinite runner développé sur Unity.

/ Space Invaders

Space Invaders est une reconstruction avec des changements du jeu original, développé sur Unity.
