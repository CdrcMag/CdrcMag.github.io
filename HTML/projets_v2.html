Projets

/ Pac Human

Pac Human est un jeu qui est né de la collaboration entre Ludus Académie et Athéo Ingénierie. Ce jeu grandement inspiré de Pac-Man a été offert aux clients d'Athéo, avec l'ambition de créer un tournoi pour le plus haut score.

/ Repentance

Repentance est un platformer 2D. On y incarne un fantôme cherchant à réparer les erreurs commises lorsqu'il était vivant. Repentance a été développé sur Unity.

/ Car Patrol

Car Patrol est un jeu mobile dans lequel vous incarnez un poussin cherchant à sauver ses oeufs sur la route. Dans ce projet j'ai occupé le rôle de designer et développeur.

/ Octogon

Octogon a été un projet mené lors du confinement 2020 en une journée. C'est un jeu mobile développé sur Unity en une journée. C’est un jeu de scoring, dans lequel il faut se déplacer pour attraper les points malgré la vitesse augmentant graduellement.

/ Micro Fun

Micro Fun est un jeu réalisé dans le cadre d'un défi ayant pour ambition de proposer de des pistes de reflexion rendant "fun" la mécanique d'appuyer sur un bouton.

/ Glow In Nightmare

Glow in nightmare a été développé pour la Mix And Game Jam 2020. C'est un jeu mélangeant le genre horreur avec le rail shooter. Réalisé avec Bastien Prob.

/ Leaf

Leaf est un petit jeu de reflexes fait en game jam en Septembre 2020. Il a été réalisé avec l’ambition d’implémenter une structure globale modulable et l'application de divers design patterns (observer/singleton/object factory).

/ Splash

Splash est un serious game platformer en 2D développé sur Unity, visant à introduire le grand public au notions d'algorithmie.

/ Roll Boll

Roll Boll est un jeu mobile de type infinite runner développé sur Unity. On y incarne un boule se déplaçant pour éviter des obstacles et gagner des points. Il a été publié en Juin 2019 sur le Google Store.

/ Cyber Ball

Cyber Ball a été mon projet de 3ème année développé sur Unity. C’est un jeu de type infinite runner, aux environnements « synthwave ». Le jeu a été pensé pour être joué avec la Kinect 2.0 sur une borne d’arcade.

/ Crash Wave

Crash Wave est une démo développée sur Unity en une semaine en Décembre 2019. C'est un jeu de course en multi-joueurs, utilisant la web-cam pour les déplacements. Les joueurs s'inclinent verticalement pour se déplacer. Aucun input clavier ou souris n'est requis.

/ Brewery Tycoon

Brewery Tycoon est une démo d'un serious game développé sur Unity. On y incarne un robot cherchant à commencer son business de fabrication de bière. On y apprend les étapes de base de la création de bière.

/ Space Invaders

Space Invaders est une reconstruction avec des changements du jeu original, développé sur Unity.

/ Sorciers Vs Vikings

Sorciers Vs Vikings est un visual novel développé sur Ren'py. Il visait à proposer une expérience narrative à plusieurs embranchements.

/ Minotaurus

Minotaurus est un jeu de plateau développé en game jam.
